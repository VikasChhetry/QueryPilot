# QueryPilot - Talk to your database. Safely.

![QueryPilot Logo](assets/icon.png)

QueryPilot is a local, AI-powered desktop assistant that allows you to interact with your MySQL databases using natural language. No more complex SQL queries – just ask in plain English, preview the generated SQL, and execute with confidence.

## Features

*   **Natural Language to SQL:** Convert your English questions into accurate MySQL queries.
*   **Local-First & Secure:** Your data never leaves your system. All queries and AI processing happen locally on your machine, ensuring maximum privacy and security.
*   **SQL Preview:** Always see the generated SQL query before execution, giving you full control and peace of mind.
*   **Intuitive Interface:** A clean and user-friendly interface designed for efficiency.

## Installation

To install QueryPilot, please download the latest installer from our [GitHub Releases page](https://github.com/VikasChhetry/QueryPilot/releases).

1.**Download Ollama from: https://ollama.com/download**
    Install Ollama for Windows
    
    Open Command Prompt / PowerShell
    
    Run:
    **ollama pull llama3.1**
    
    Make sure Ollama is running:
    **ollama serve**
    
    or 
    **open the Ollama app**
    
2.  **Download:** Choose the appropriate installer for your operating system (e.g., `QueryPilot Setup X.X.X.exe` for Windows).
3.  **Run Installer:** Execute the downloaded installer and follow the on-screen instructions.

## Usage

1.  **Launch QueryPilot:** After installation, launch the application.
2.  **Connect to Database:** On the initial screen, you will be prompted to enter your MySQL database connection details (host, port, username, password, database name).
3.  **Start Querying:** Once connected, you can type your questions in natural language into the input field.
    *   Example: "Show me all users who registered last month."
    *   Example: "What are the top 5 products by sales revenue?"
4.  **Preview and Execute:** QueryPilot will generate the corresponding SQL query. Review the SQL query to ensure it matches your intent, then click "Execute" to run it against your database.
